# FlowZen AI

FlowZen AI is an AI-powered event management and crowd monitoring platform for modern venues, festivals, and high-capacity spaces.

## Description

AI-powered event management and crowd monitoring platform that helps teams control capacity, optimize bookings, and keep events safe with real-time alerts.

## Features

- Smart crowd alerts with automatic high-capacity warnings
- Booking management for seamless ticket handling
- Real-time capacity tracking with color-coded crowd levels
- Accessible dashboard with responsive, production-ready styling

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- Framer Motion

## Live Demo

https://flowzen-ai.vercel.app

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser:

```bash
http://localhost:3000
```

Build for production:

```bash
npm run build
npm start
```

## Testing

Run Jest tests:

```bash
npm test
```
