# FlowZen AI 🎪
AI-powered Smart Event & Crowd Management Platform

## Features
- Smart crowd monitoring with real-time alerts
- AI gate assignment & QR tickets
- VIP/Artist private access panel
- Smart stall management
- Google Maps, Calendar integration
- Weather-based gate recommendations
- Emergency SOS system

## Tech Stack
Next.js, TypeScript, Tailwind CSS, Firebase

## Live Demo
https://flowzen-ai.vercel.app

npm run dev
